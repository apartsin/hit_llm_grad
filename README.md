# Large Language Models and Agentic AI: Graduate Course

13-week graduate course based on the book [Building Conversational AI with LLMs and Agents](https://llmbook.apartsin.com).

- **Syllabus**: [syllabus.html](https://apartsin.github.io/hit_llm_grad/syllabus.html)
- Weeks 5, 8, and 13 are student presentation sessions (project proposal, interim, final).
- Research-oriented student projects delivered as documented GitHub repositories.
